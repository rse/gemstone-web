Gemstone
Best of Breed for JavaScript
Single-Page-Apps...
Gemstone is a build-time framework for scaffolding and building JavaScript Single-Page-Apps (SPA). In a Best-of-Breed approach, it leverages the main build-tools Yo, Bower, Grunt, Traceur and LESS and the main run-time components ComponentJS, jQuery and RactiveJS.
Step 1: Initial Project Setup 

$ npm install gemstone
$ mkdir example
$ cd example
$ yo gemstone
$ vi gemstone.yaml
Install Gemstone with NPM, create a fresh project directory and then create the initial Gemstone project configuration. Finally, edit the Gemstone project configuration accordingly.

Step 2: Scaffold Build Environment 

$ yo gemstone:env --app-name="Example"
Now, scaffold the build environment according to the previously setup project configuraton.

Step 3: Add Composites and Widgets 

$ yo gemstone:widget "Login"
$ yo gemstone:composite "Panel"
Add ComponentJS MVC/CT-based widgets and composites.

Step 4: Build Application 

$ npm install
$ grunt
Locally install the Grunt build tool dependencies and run it, which in turn will run the Bower package manager and install all third-party libraries. Finally, the initial application will be built.

Step 5: Run Application 

$ grunt server
Run the application via http://localhost:8080/
